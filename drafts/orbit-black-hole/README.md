# Обратная сторона — черновик 01

Первая версия сайта-визитки для медиа «Обратная сторона». В центре сцены находится чёрная дыра, в которую по орбитальной траектории уходят картина Ван Гога, портрет и книга Достоевского, статуя Родена и газета The Times.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
